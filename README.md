

# Dashboard de Produtividade - Pomodoro

* Python
* SQLite
* OOP (Programação Orientada a Objetos)
* Streamlit

## Funcionalidades

* Registro de atividades com nome, hora de início e duração.
* Tabela interativa com todos os registros do mês (com opção de excluir entradas).
* Gráficos mostrando a quantidade de sessões realizadas por dia.
* Exibição do total de sessões e total de horas trabalhadas.
* Barra de progresso visual indicando o tempo restante da sessão.
* Reprodução automática de ruído branco durante cada sessão, para ajudar na concentração.
