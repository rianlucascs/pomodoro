

# Dashboard de Produtividade - Pomodoro

* Python
* SQLite
* OOP (Programação Orientada a Objetos)
* Streamlit

## Funções Interativas

* Registra: nome da atividade, hora de início e duração da atividade.

## Visualização

* Tabela com os registros do mês (com opção de apagar registros na tabela).
* Gráfico com a quantidade de sessões realizadas por dia.
* Total de sessões.
* Total de horas das sessões.
* Progresso do tempo da sessão.

## Som

* Ao criar uma nova sessão, é iniciado um som de *ruído branco* que dura todo o tempo da sessão.
